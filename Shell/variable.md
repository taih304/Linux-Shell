**variable**

to create a new variable

```
a=8
```

Note: if you type a = 9, or you leave space between the "=",this will be an error

error: command not found

**display**

```
echo a
```
```
result: a
```

```
echo $a
```
```
result: <value of a>
```

**string**

```
a = "string";
```

**()**
```
a=0
a=$((a+9))

echo $a
```

result: 9

REMEMBER: there is no () => WRONG
