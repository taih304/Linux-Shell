``Ctrl+D`` logs out of the interface (quite similar to Ctr+C to exit)

``which``: locate where a command is stored. 

**E.g**: ``which ls``: ``/bin/ls``

### For space

To work with a file containing space, like ``"Folder 1"``:

* Add ``" "`` in the command: E.g ``cd "Folder 1"``
* Add ``\`` in the command: E.g ``cd Folder\ 1/"``
