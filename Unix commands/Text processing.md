### wildcard

``*`` : alias

``?`` : every "?" stand for one character

### wc

``wc``: word count, used to count words in a file

```shell
$ wc -w filename #print total word in filename
```

**Result**: ``10 filename``

(``filename`` has 10 words)

```shell
$ wc -w test.sh
```
**Result**: ``10``

* ``wc -l``: print newline count
